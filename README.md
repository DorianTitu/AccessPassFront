# Screenshot Capturer

Aplicación web para capturar pantalla construida con React, Vite y TypeScript.

## Instalación

```bash
npm install
npm run dev
```

Disponible en http://localhost:5173/

## Scripts

- `npm run dev` - Servidor de desarrollo
- `npm run build` - Build para producción
- `npm run preview` - Preview de la build

## Tecnologías

- React 19
- Vite
- TypeScript
- html2canvas
- CSS3

## Estructura

```
src/
├── components/
├── pages/
├── services/
├── data/
├── App.tsx
└── main.tsx
```

## Uso

Presiona el botón para capturar la pantalla. La imagen se descarga automáticamente.

## Build

```bash
npm run build
```

Genera carpeta `dist/` lista para producción.
